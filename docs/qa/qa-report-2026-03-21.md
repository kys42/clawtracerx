## 개선 제안

1. **세션 카드를 `<a>` 태그로 감싸기** — 현재 `onclick` 방식은 새 탭(Cmd+Click) 불가, 접근성(keyboard focus) 부족
2. **검색 결과 카운트 표시** — "3 of 50 sessions" 피드백
3. **키보드 단축키 가이드** — `?` 키로 단축키 모달
4. **세션 상세 브레드크럼** — `Sessions > aki > addd575a-d04`
5. **다크/라이트 테마 토글** — 프레젠테이션/문서화 용도
